# dwgrambot
> Hosted bot: [@dwgrambot](https://t.me/dwgrambot)
## Install on Linux
NOTE: you need to have git installed.
```bash
git clone https://github.com/ShiSHcat/dwgrambot.git
mv config.example.json config.json
```
Edit config.json with the right data.
